## Requirement
Install [Node.js](https://nodejs.org/en/).

## Usage

### Getting Started

Fork this repository to your own Github account.

### Setup
Install the dependencies.
```bash
$ npm install
```

### Development
Run the local webpack-dev-server & open [http://localhost:8080/](http://localhost:8080/)
```bash
$ npm run dev
```
